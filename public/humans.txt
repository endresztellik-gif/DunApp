/* DATA SOURCES */

Meteorology: OMSZ (Hungarian Meteorological Service)
Water Level: VízÜgy (Hungarian Water Authority)
Drought Monitoring: OVF + HUGEO
Groundwater: VízÜgy Wells Database

/* MODULES */

1. Meteorology Module (Meteorológia)
   - 4 cities: Szekszárd, Baja, Dunaszekcső, Mohács
   - Real-time weather data
   - 7-day forecast
   - Radar imagery

2. Water Level Module (Vízállás)
   - 3 stations: Baja, Mohács, Nagybajcs
   - Danube water level monitoring
   - Historical data charts
   - Flood alerts

3. Drought Module (Aszály)
   - 5 monitoring locations
   - 15 groundwater wells
   - Soil moisture data
   - Drought index calculations
   - Interactive maps

/* LOCATIONS */
